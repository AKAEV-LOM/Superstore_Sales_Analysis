# Анализ продаж розничной сети Superstore

Проект по полному циклу обработки данных: от очистки в Python до интерактивного дашборда в Power BI.

##  Стек технологий
* **Python (Pandas):** Очистка данных и ETL-процессы.
* **SQL (PostgreSQL):** Аналитические запросы и расчет бизнес-метрик.
* **Power BI:** Визуализация данных и создание интерактивных отчетов.

##  Основные этапы
1. **Очистка данных:** Обработаны пропуски, изменены форматы дат для корректного анализа.
2. **SQL Аналитика:** Написаны запросы для выявления топ-клиентов, анализа продаж по штатам и динамики выручки.
3. **Визуализация:** Построен дашборд с фильтрацией по регионам, сегментам и датам.

##  Результаты
* Общая выручка: **$2.26M**.
* Средний чек: **$230.77**.
* Самая прибыльная категория: **Technology**.

##  Рекомендации для бизнеса
1. Усилить фокус на категорию **Technology** (лидер выручки).
2. Оптимизировать запасы в штатах **California** и **New York** перед пиковыми сезонами (Q4).
3. Внедрить систему Cross-sell для увеличения среднего чека.
